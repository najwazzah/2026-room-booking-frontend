# RoomBooking Frontend

Frontend aplikasi Sistem Peminjaman Ruangan Kampus.  
Dibangun dengan **React + TypeScript**.

---

## Cara Menjalankan

1. Clone repo frontend:
   ```bash
   git clone https://github.com/username/RoomBookingFrontend.git
   cd RoomBookingFrontend
2. Install dependencies:
   npm install
3. Jalankan frontend:
   npm start
4. Buka di browser:
   http://localhost:3000
